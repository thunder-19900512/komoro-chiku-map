# こもろ地区ガイド（公開層・Phase A）

小諸市の10地区を知る公開マップ。仕様: `../shutsuba/仕様書_地区マップアプリ_v1.md`

## ⚖️ 憲法（要旨）
- **公開層。非公開データ（活動ログ・地区メモ・個人情報）をこのフォルダに置かない**
- 「集票」等の表現禁止／発信者明記（山田ゆうじ）／区界は「おおよそ」と必ず注記

## 使い方
```bash
./venv/bin/python build.py          # data/estat_shape → docs/ を再生成
cd docs && python3 -m http.server 8765   # ローカル確認 http://localhost:8765
```

## 構成
- `data/estat_shape/` … e-Stat 令和2年国勢調査 小地域境界（小諸市 20208）
- `data/対応表_要確認リスト.csv` … 地区対応が未確認の小地域（**本人レビュー待ち**）
- `build.py` … 対応表（コード内 `D`）＋地区紹介（`CHIKU_INFO`）→ docs/ を生成
- `docs/` … 静的サイト一式（GitHub Pages公開想定。index.html + data.geojson + chiku_info.json）

## 収録内容（2026-07-16 地域別構想＋年齢構成反映済み）
- 10地区×〔将来像・特徴・人口世帯・**高齢化率・年齢3区分バー**・行政区・住民の声/論点〕
- 塗り分け3モード: 地区色／人口／**高齢化率**（小地域別）
- 出典=都市計画マスタープラン地域別構想（H28）＋e-Stat（境界・人口世帯＋**第3表 年齢5歳階級 statInfId=000032163380 の再掲3区分**）
- 区数は全68区（北大井15区=MP p77で裏付け）。要確認8小地域は行政区マップからざっくり割当済（⚠️表示は維持）
- ⚠️ 年齢集計の注意: 5歳刻み列の自前合算は**再掲列を二重カウントする**ため、必ず「（再掲）15歳未満/15～64歳/65歳以上」列を使うこと（1敗済み）
